## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:gabrielyamazato/solar-system.git
```

Entre no diretório do projeto

```bash
  cd solar-system/
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
